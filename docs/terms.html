Description of Service

IrriGo is a remote motor and pump control application for agricultural irrigation systems. The App communicates with physical IoT motor control hardware via AWS IoT MQTT to enable remote monitoring and control of up to 6 motors per site.

Hardware Requirement

The App requires a physical IrriGo motor control device, purchased separately from AbhiTech. Each device includes a QR code for site setup. The App is not functional without the companion hardware device and a valid site QR code.
